# LinDream 插件仓库

## 插件安装指南

### 如何添加插件

1. 将解压后的插件文件夹放入 `plugin` 目录下
2. 确保插件目录包含以下必要文件：
   - `main.py` - 插件主文件
   - `requirements.txt` - 依赖文件（如果需要）
   - `metadata.json` - 插件元数据（如果需要）

### 插件放置方法

1. 解压插件，放置在对应目录
   ```
   LinDream/plugin/
   ```

2. 在插件目录中放置插件文件：
   ```
   /plugin/插件名称/
   ├── main.py          # 插件主文件
   ├── requirements.txt # 依赖文件
   ├── README.md        # 插件说明（可选）
   └── 使用指南.txt     # 使用指南
   ```

### 插件结构要求

- 每个插件必须是一个独立的目录
- 插件主文件必须命名为 `main.py`
- 插件必须包含 `on_message` 函数处理消息
- 插件可以包含 `on_load` 函数进行初始化
- 插件可以包含 `on_command` 函数处理命令

### 激活插件

1. 将插件放置到正确位置后
2. 重启 LinDream 机器人
3. 插件将会自动加载

### 注意事项

- 确保插件名称不与其他插件冲突
- 插件目录名应使用英文或拼音
- 插件代码需要符合 LinDream 插件开发规范
- 插件放置时请确保目录权限正确
